# 1.产品特点

●  支持802.11b/g/n 无线标准

●  支持STA/AP工作模式

●  支持无线和远程升级固件，提供无线批量配置工具

●  可提供SDK开发包，支持二次开发

●  可选板载PCB天线或外置天线接口（I-PEX）

●  超小尺寸：23.1mm x 32.8mm x 3.5mm

●  产品通过FCC/CE/TELEC标准认证
